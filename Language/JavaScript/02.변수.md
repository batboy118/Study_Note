## 1. 변수의 사용

변수가 정적이지 않기 때문에, 변수에 어떠한 타입의 값이라도 넣을 수 있다.

```javascript
var a = 'coding', b = 'everybody';
a = 3
b = 4
console.log(a + b) // 7
```

## 2. 변수의 효용

```javascript
//변수를 쓰기 전

alert(100+10);
alert((100+10)/10);
alert(((100+10)/10)-10);
alert((((100+10)/10)-10)*10);

// 변수 사용 후

a = 100;
a = a + 10;
alert(a);
a = a / 10;
alert(a);
a = a - 10;
alert(a);
a = a * 10;
alert(a);
```

## 3. 변수의 선언

변수의 앞에 특정 예약어를 써주지 않아도 자동으로 변수를 선언해 줄 수 있지만, 좋지 않은 방법이다. 변수를 선언하는 방법에 대해서 알아보자.

var : 이전에 선언되었던 변수를 재선언 할 수 있음

```javascript
var a = 1;
var a = 2;
```

let : 동일한 변수명을 재선언 할 수 없음

```javascript
let a = 1;
//let a = 2; 불가
```

const : 상수 선언
